-----------Interrupted method-----------------

- It is used to interrupt an executing thread.

- interrupt() method will only work when the
thread is in sleeping or waiting state.
(sleep() or wait() )


- If thread is not sleeping or waiting state then
calling an interrpt() method then it will perform
normal method or normal behaviour.

-  when we use interrupt() method it always throws exception
InterruptedTHreadException.
