----------synchronization-------
=> synchronization means at a time ine thread execute , other thread depend on current execution thread

-If we not use synchronazation ,problem occurs
1)data Inconsistency
2)Thread Interference

- If we use synchronization
1)No data inconsitency problem
2)No data intereferences


- if we use synchronization , it increase the waitiing time period of thread
and application performance down ,(slowly working)

- In latest version , solve this performance issue ,that why java introduce , java.util.concurrent.

- To overcome synchronization disadvantages , java provides one package
java.util.concurrent.
