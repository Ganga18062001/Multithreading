----Thread priority--------
-JVM provides the priority to each thread and according to these
priority jvm allocates the processor.

- If priority is same , jvm will be decide internally.
-priority are represent in the form of integer value which
range is from 1-10.
=> 1 = MIN_Priority
=> 5 = NORM_Priority
=> 10 = MAX_Priority

-below are not priority
=> 0 , <1 , >10
- MINIMUM_Priority
- LOW_Priority
-MINIMUM_Priority
-HIGH_Priority

-above all are not priority
---methods of priority----
-public final void setPriority(int value)
-public final int getPriority()
